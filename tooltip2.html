.tool_tip() a jQuery method


Rose Wiki Rose

The spelling using the underscore is intended to distinguish Horn's "tool_tip()" method (in mylib.js) from the large packages such as qTip2, Tooltip, etc. See CodeGeekz.

This jQuery method requires elements in pairs of "trigger" and "tooltip". jQuery's .next() function merely selects whatever element is next and displays it as the tooltip. The trigger and the tooltip elements must be chosen appropriately For instance, in text passages use intertextual tags like the "em" and "span" tags so as not to interrupt paragraph formation.

Hover over the trigger to expose the tooltip. Hover over the exposed tooltip to keep it in view.

Change the values of "margin-left" and "margin-top" (of the tooltip style) to change the position of the tooltips relative to the targets. A positive "margin-left" moves the tooltip to the right
and a positive "margin-top" moves the tooltip down.
Negative values reverse the direction.

The tooltip shows above the target if there is room on the screen. Otherwise it shows under the target.
